Mac配置zsh

发表于 2018-08-23

安装zsh

brew install zsh

配置zsh

nano ~/.zshrc

复制以下内容到.zshrc

为root用户修改默认shell为zsh

chsh -s /bin/zsh root
为当前用户修改默认shell为zsh
chsh -s /bin/zsh
or
chsh -s `which zsh`
恢复命令

chsh -s /bin/bash
